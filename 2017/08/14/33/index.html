2017下半年书单列表

Posted on 08-14-2017   |   In 杂文

筛选自公众号 一晌贪欢 的微信推送

社科类

  1. 《狂热分子》 - 有助于理解社会上大多数群体行为以及我们为什么那么喜欢群体运动
  2. 《中国文化的深层结构》 - 《巨婴国》的高级讲法
  3. 《沉默的大多数》- 王小波
  4. 《黄金时代》- 王小波
  5. 《乡土中国》- 农村运作的逻辑
  6. 《娱乐至死》 - 媒介之惑
  7. 《社会心理学》
  8. 《影响力》 - 管理即影响他人来为你做事，如何提高影响力的实用技巧

传记类

  1. 《人类群星闪耀时》 - 被打了鸡血的传记

实用类

  1. 《穷爸爸富爸爸》 - 思维转变
  2. 《自控力》- 我最需要的一样东西
  3. 《穷查理宝典》- 思维转变
  4. 《大问题：简明哲学导论》 - 未经审视的生活是不值得过的
  5. 《进化心理学》

思想

  1. 《追寻生命的意义》 - 我们并不是为了逃避苦难一味追求快乐，而是苦难要有意义
  2. 《理想国》- 洞穴中的人
  3. 《论自由》- 政治上对自由的最佳定义
  4. 《人生的智慧》- 叔本华
  5. 《禅与摩托车维修技术》

历史

  1. 《邓小平时代》
  2. 《华夏边缘》- 中华民族是如何形成的
  3. 《中国史学入门》 - 极简中国历史通论
  4. 《昨日的世界》- 远去的欧洲
